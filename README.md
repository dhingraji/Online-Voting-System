# Online Voting System 🗳️

## 📌 Description
The Online Voting System is a web-based application that allows users to cast votes securely through an online platform. It is designed to reduce manual effort, improve accuracy, and provide a transparent voting process.

## 🚀 Features
- User authentication (Voter & Admin)
- Secure login system
- Cast vote online
- One vote per user
- Admin dashboard to manage candidates and results
- Real-time result calculation

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## 🖥️ System Roles
- **Admin**
  - Add candidates
  - View results
- **Voter**
  - Login
  - Cast vote
  - Logout

## ▶️ How to Run the Project
1. Download or clone this repository
2. Place the project folder inside `htdocs` (XAMPP)
3. Start **Apache** and **MySQL**
